# StackTower 3D

> A 3D Hyper-casual game built with **Unity 6**. The goal is to stack moving blocks as high as possible. The game features a dynamic slicing mechanic where placing a block imperfectly cuts off the overhanging part, making the tower progressively thinner and harder to stack.
